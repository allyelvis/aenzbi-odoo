# Aenzbi-Odoo

## Setup

1. Clone the repository:
   ```
   git clone https://github.com/allyelvis/aenzbi-odoo.git
   ```

2. Navigate to the project directory:
   ```
   cd aenzbi-odoo
   ```

3. Set up the environment:
   ```
   ./setup.sh
   ```

4. Build and run the Docker containers:
   ```
   docker-compose up --build
   ```
